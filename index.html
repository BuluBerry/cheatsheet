CSS Properties CheatSheet

Background

Property Description Values
background-color sets the background color of an element "color"
background-image sets one or more background images on an element none, url: ("")
background-position sets the initial position for each background image. The position is relative to the position layer set by background-origin top, bottom, left, right, center, "percentage", "length"
background-repeat sets how background images are repeated. A background image can be repeated along the horizontal and vertical axes, or not repeated at all repeat-x, repeat-y, repeat, space, round, no-repeat


Border

Property Description Values
border sets an element's border color, style, width
border-color sets the color of an element's border "color t/r/b/l", inherit, initial, revert, revert-layer, unset
border-width sets the width of an element's border top, bottom, left, right, center, thin, medium, thick
border-style sets the line style for all four sides of an element's border none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset
border-radius rounds the corners of an element's outer border edge top, right, bottom, left, "percentage", "length"


Floats

Property Description Values
float places an element on the left or right side of its container, allowing text and inline elements to wrap around it left, right, none, inline-start, inline-end
clear sets whether an element must be moved below (cleared) floating elements that precede it none, left, right, both, inline-start, inline-end
overflow sets the desired behavior for an element's overflow visible, hidden, clip, scroll, auto, hidden visible


Fonts

Property Description Values
color sets the foreground color value of an element's text and text decorations, and sets the currentcolor value color name, hex, rbg(a), hsl(a)
font sets all the different properties of an element's font family, size, stretch, style, variant, weight
font-family specifies a prioritized list of one or more font family names and/or generic family names for the selected element "font name", serif, sans-serif, monospace, cursive, fantasy, system-ui, ui-serif, ui-sans-serif, ui-monospace, ui-rounded, emoji, math, fangsong
font-style sets whether a font should be styled with a normal, italic, or oblique face from its font-family normal, italic, oblique, oblique 10deg
font-weight sets the weight (or boldness) of the font. The weights available depend on the font-family that is currently set normal, bold, lighter, bolder, number "100 - 900"
font-variant allows you to set all the font variants for a font. small-caps, common-ligatures small-caps
font-size sets the size of the font xx-small, x-small, small, medium, large, x-large, xx-large, xxx-large
line-height sets the height of a line box. It's commonly used to set the distance between lines of text normal, "number multiplies font size"


Margin & Padding

Property Description Values
margin sets the margin area on all four sides of an element margin-top/right/bottom/left
padding sets the padding area on all four sides of an element at once padding-top/right/bottom/left


Positioning

Property Description Values
display sets whether an element is treated as a block or inline element and the layout used for its children, such as flow layout, grid or flex block, inline, inline-block, flex, inline-flex, grid, inline-grid, flow-root
position sets how an element is positioned in a document. The top, right, bottom, and left properties determine the final location of positioned elements static, relative, absolute, fixed, sticky
z-index sets the z-order of a positioned element and its descendants or flex items auto, integer


Width & Height

Property Description Values
width sets an element's width "length", "percentage"
height specifies the height of an element "length", "percentage"
min-width sets the minimum width of an element "length", "percentage"
max-width sets the maximum width of an element "length", "percentage"
min-height sets the minimum height of an element "length", "percentage"
max-height sets the maximum height of an element "length", "percentage"
